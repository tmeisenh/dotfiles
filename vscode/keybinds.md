## Common keybindings

### accessing the left menu

open file explorer

```
command shift e
```

open file search

```
command shift f
```

open source control

```
control shift g
```

hide left menu

> (has issues in some file types, like md)

```
command b
```

### General

fuzzy file

```
command p
```

fuzzy program/command

```
command shift p
```

focus pane

```
command 0, 1, 2, etc
control H, L
```

split new pane

> conflicts with 1password

```
command \
```

move pane right/left

```
command control left-arrow
command control right-arrow
```

jump block down (extension)

```
control shift j
```

jump block up (extension)

```
control shift h
```
